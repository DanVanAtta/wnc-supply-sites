Uploading Needs

WNC-Supply-Sites.com is deeply integrated with R4 to track inventory needs and help dispatch.

The process starts when inventory is updated on WNC-Supply-Sites.com.

Below is a video of how the data flow looks, on the right are updates being made on WNC-Supply-Sites.com and the left is a visualization of the real time data flowing into R4.

Needs Dashboard

The uploaded inventory data lands in the R4 Needs-Dashboard where a dispatcher can see how many needs there are, and where.

Needs Matching

Clicking the Needs-Matching link takes a dispatcher to WNC-Supply-Sites needs matching table. The table shows which sites have needed supplies.

Setting up a Delivery

After needs are matched, a dispatcher can then start setting up a delivery. This is done in the R4 system. The dispatcher creates the delivery, sets up the 'from' and 'to' sites, and adds items to the delivery.

Assigning a Driver

Now that we have a delivery set up, next we need to arrange transportation. To do this, we need to find and contact drivers from the driver listing. (This part is currently a bit laborious). Once done, we have a delivery set up!
